# 우쿨렐레 코드 찾기 🎵

가수 이름과 곡명을 입력하면 AI가 그 곡의 **우쿨렐레 코드 악보**(가사 위에 코드 + 운지표)를 만들어주는 모바일용 웹사이트.

- **서버 0원** — 순수 정적 사이트. 브라우저가 Anthropic API를 직접 호출.
- 검색 기록 · 즐겨찾기 (브라우저에 저장)
- 곡에 쓰인 코드의 우쿨렐레 운지 다이어그램 자동 표시
- 글씨 크기 조절

## 준비물: Anthropic API 키

이 앱은 AI(Claude)로 코드를 생성하므로 **본인 API 키가 필요**합니다.

1. https://console.anthropic.com/settings/keys 에서 키 발급 (`sk-ant-...`)
2. 앱 우측 상단 **설정(⚙)** → API 키 입력 → 저장
3. 키는 브라우저(localStorage)에만 저장됩니다. Anthropic API 호출 외에는 어디로도 전송되지 않아요.

> 비용: 검색 1회당 보통 몇 원 수준. 모델은 설정에서 바꿀 수 있어요(기본 Opus 4.6, 더 저렴한 Haiku/Sonnet 선택 가능).

## 로컬에서 실행

그냥 `index.html`을 브라우저로 열어도 동작합니다. 일부 브라우저에서 막힐 경우 간단한 정적 서버로:

```bash
cd ukulele-chord-finder
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 무료 배포 (휴대폰에서 쓰기)

아래 중 아무거나 — 모두 무료, 서버 유지비 없음:

- **GitHub Pages**: 이 폴더를 깃 저장소로 올리고 Settings → Pages에서 활성화
- **Cloudflare Pages / Netlify / Vercel**: 폴더를 드래그 앤 드롭으로 업로드

배포 후 휴대폰 브라우저에서 열고 "홈 화면에 추가"하면 앱처럼 쓸 수 있어요.

## 파일 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css` | 스타일(모바일 세로 기준) |
| `app.js` | 검색·확인·생성 흐름, 기록/즐겨찾기, API 호출 |
| `chords.js` | 우쿨렐레 코드 운지 다이어그램 SVG 렌더러 |

## 동작 방식

1. 가수+곡명 입력 → AI가 곡 식별 → "이 곡이 맞나요?" 확인
2. 확인 → AI가 전체 가사 + 코드 + 운지를 JSON으로 생성
3. 가사 위에 코드를 얹은 악보 + 운지표 렌더링

운지(프렛 번호)를 AI가 직접 생성하므로 별도의 코드 데이터베이스가 필요 없습니다.
